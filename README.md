# YouTube Lite

A lightweight WebView-based YouTube client app with ad-blocking and background playback.

[![Download APK](https://custom-icon-badges.herokuapp.com/badge/-Download%20APK-2ea44f?style=for-the-badge&logo=download&logoColor=white)](https://github.com/munnakumar678997/YouTube-Lite/releases/latest/download/youtube-lite-signed.apk)

Click the button above to download the latest signed APK (built automatically from `main` by GitHub Actions).

## Features

* Ad blocker (blocks Google ads, doubleclick, pagead)
* Background playback (audio keeps playing when screen is off)
* Fullscreen video support
* Notification media controls (play/pause/next/previous)
* Lightweight and minimal

This is an educational project showcasing JavaScript injection into a WebView. It is not affiliated with or endorsed by YouTube/Google.
